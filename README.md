📊 Exploratory Data Analysis (EDA) Project
🚀 Overview

This project performs Exploratory Data Analysis (EDA) on a structured sales dataset to uncover meaningful insights, identify trends, and support data-driven decision-making.

The analysis focuses on understanding sales performance, customer behavior, and product efficiency across different dimensions.

📂 Dataset Description

The dataset is organized using a star schema model:

🔹 Fact Table
fact_sales → Contains transactional sales data
🔹 Dimension Tables
dim_customers → Customer-related details 👤
dim_products → Product-related details 🛍️
🎯 Objectives
📈 Analyze overall sales performance
👥 Understand customer distribution and contribution
🏆 Identify top-performing and low-performing products
🌍 Analyze sales distribution across countries
📊 Evaluate category-wise revenue trends
